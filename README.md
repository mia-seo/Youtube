# Youtube

<a href="https://developers.google.com/youtube" target="_blank">Youtube open API</a>를 이용한 Youtube 클론코딩

### 기술 스택 ⚒️

- React.js
- tailwindcss
- Github

### 시작 방법 ✨

```
$ yarn install
$ yarn start
```
